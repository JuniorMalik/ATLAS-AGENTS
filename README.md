# ATLAS-AGENTS
ATLAS: Sistema Avançado de Inteligência Multi-Agente para pesquisa autônoma. Desenvolvido com LangGraph, Python e Gemini 1.5 Flash. Utiliza orquestração cíclica entre agentes de Pesquisa, Redação e Crítica para garantir relatórios técnicos de alta precisão. Integra Tavily Search para mineração web em tempo real e uma CLI estilizada e profissional.
